# 私を構成する9枚のカード

MTGカードを選んで、あなたを表す9枚を共有しよう。

## サイトURL

https://soichirow.github.io/my-9-cards-mtg/

## つかいかた

1. 空きスロットをタップして検索モーダルを開く
2. カード名を入力して検索（日本語・英語対応）
3. 検索結果からカードを選択してスロットに配置
4. 9枚すべて選んだら共有！

### 便利な機能

- **ドラッグ&ドロップ** — カードの並び順を入れ替え
- **他Ver** — 同じカードの別イラスト・別セットに変更
- **リンクをコピー / リンクで開く** — 選んだカードをURLで共有
- **画像で保存** — 3x3のカード画像を保存（保存時に名前を入力可能）
